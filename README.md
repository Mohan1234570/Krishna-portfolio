## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

## Customization

### Update Personal Information

1. **Contact Information**: Update contact details in `src/components/Contact.jsx`
2. **Social Links**: Update social media links in `src/components/Contact.jsx` and `src/components/Hero.jsx`
3. **Experience**: Modify experience data in `src/components/Experience.jsx`
4. **Projects**: Update projects in `src/components/Projects.jsx`
5. **Skills**: Adjust skills and proficiency levels in `src/components/Skills.jsx`

### Styling

- Global styles: `src/index.css`
- Component styles: Each component has its own CSS file in `src/components/`

## Sections

1. **Hero**: Eye-catching introduction with contact information
2. **About**: Professional summary and key features
3. **Experience**: Timeline of work experience with achievements
4. **Projects**: Featured projects with technologies and highlights
5. **Skills**: Technical skills with animated progress bars
6. **Education**: Educational background
7. **Achievements**: Awards and certifications
8. **Contact**: Contact form and social links
